# 3DCV_final
## Dataset
We use real-world dataset from [NeRF-DS](https://github.com/JokerYan/NeRF-DS/releases/tag/v0.1-pre-release)
We organize the datasets as follows:
```shell
├── data
│   | NeRF-DS
│     ├── as
│     ├── basin
│     ├── bell
│     ├── cup
│     ├── plate
│     ├── press
│     ├── sieve
```
